Bootstrap Quiz

Objectives

  1. Center the Bootstrap Quiz Header
  2. Change the background color to Slate Gray
  3. Create a new row and create 3 column 4s with a card in each column and make the cards evenly spaced
  4. Add a different image to each card and add lorem ipsum to each card
  5. Create a paragraph of lorem ipsum and change the font to a Google Font of your liking
  6. Add a button and use WickedCss to make the button spin
